Understanding Log Prefixes

[AUTH]

Student node authentication. Occurs when you sign in or refresh your API Bearer Token.

[GRAD]

Grading Kernel activity. Triggered when you submit a lab code packet for evaluation.

[WARN]

Non-critical system alerts, such as high latency in a specific regional learning node.

Why Read Logs?

In high-level software engineering, reading logs is a core skill. It allows you to trace errors, monitor performance, and verify that your deployments are behaving as expected in production.
